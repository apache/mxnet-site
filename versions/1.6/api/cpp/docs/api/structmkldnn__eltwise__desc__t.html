mkldnn_alg_kind_t mkldnn_eltwise_desc_t::alg_kind

The kind of eltwise algorithm. Possible values: mkldnn_eltwise_relu, mkldnn_eltwise_tanh, mkldnn_eltwise_elu, mkldnn_eltwise_square, mkldnn_eltwise_abs, mkldnn_eltwise_sqrt, mkldnn_eltwise_linear, mkldnn_eltwise_bounded_relu, mkldnn_eltwise_soft_relu, mkldnn_eltwise_logistic and mkldnn_eltwise_exp.

float mkldnn_eltwise_desc_t::alpha

Algorithm specific parameter. Accordance table:

  • mkldnn_eltwise_relu: alpha – negative slope, beta ignored
  • mkldnn_eltwise_tanh: alpha and beta ignored
  • mkldnn_eltwise_elu: alpha – negative slope, beta ignored
  • mkldnn_eltwise_square: alpha and beta ignored
  • mkldnn_eltwise_abs: alpha and beta ignored
  • mkldnn_eltwise_sqrt: alpha and beta ignored
  • mkldnn_eltwise_linear: alpha – scale, beta – shift
  • mkldnn_eltwise_bounded_relu: alpha – upper bound, beta ignored
  • mkldnn_eltwise_soft_relu: alpha and beta ignored
  • mkldnn_eltwise_logistic: alpha and beta ignored
  • mkldnn_eltwise_exp: alpha and beta ignored
