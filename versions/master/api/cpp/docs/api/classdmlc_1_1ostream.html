Detailed Description

a std::ostream class that can can wrap Stream objects, can use ostream with that output to underlying Stream

Usage example:

Stream *fs = Stream::Create("hdfs:///test.txt", "w");
dmlc::ostream os(fs);
os << "hello world" << std::endl;
delete fs;
